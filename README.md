Interactive Configurator — это консольная утилита для Windows Server, предназначенная для интерактивной настройки сложных платформ с большим количеством конфигурационных файлов.

Процесс установки описывается в виде JSON-сценария, который задаёт список параметров и вопросов для пользователя. Во время работы программа последовательно запрашивает необходимые значения, проверяет их корректность и автоматически применяет настройки ко всем нужным конфигурационным файлам из одного исполняемого файла, без ручного редактирования конфигов.

````
interactive-configurator/
├── cmd/
│   └── configurator/
│       └── main.go             # точка входа программы, запускает интерактивный CLI
│
├── internal/
│   ├── scenario/               # модели сценария и работа с JSON
│   │   ├── types.go            # ValueType и константы типов
│   │   ├── model.go            # структура Step, Scenario
│   │   └── loader.go           # загрузка сценария из JSON и базовая проверка
│   │
│   ├── validate/               # валидаторы для разных типов значений
│   │   ├── validator.go        # интерфейс Validator
│   │   └── basic.go            # реализация базовых типов Validator
│   │
│   ├── input/                  # интерактивный CLI для ввода значений
│   │   └── prompt.go           # опрос пользователя, обработка ошибок и пропусков
│   │
│   └── writer/                 # Пишет изменения
│       └── prompt.go          
│
├── examples/
│   └── scenario.json           # пример JSON-сценария для теста и демонстрации
│
├── README.md                   # описание проекта, инструкция
└── go.mod                      # Go-модуль, управляет зависимостями и пакетами
````
